Add Java Platform Wizard: Select Platform Type

Use to register Java and JavaFX platforms as well as Java ME MIDP platform-compatible SDKs or emulator platforms, such as the Java ME SDK, in the NetBeans IDE. Once you register a platform emulator, you can use project configurations to set that platform for compilation, execution and debugging.

Element Description

Select platform type

 Selects the desired emulator platform:
  • Java Standard Edition
  • Java ME CLDC platform

  • Custom Java ME CLDC Platform Emulator

    (Select this for a non-UEI-compliant emulator platform)

  • Java ME CDC Platform Emulator

Clicking Next after selecting Custom Java ME CLDC Platform Emulator takes you to the General Information page. Clicking Next after selecting a UEI-compliant platform takes you to the Platform Folders page.

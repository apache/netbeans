Adding a Docker Instance

See Also

To use a Docker Instance, you must first add a Docker instance, build the image and then run the Docker Container.


To add a Docker Instance:
  1. In the Services window, right-click on Docker.
  2. Enter the Name and URL and connect to the Docker daemon using http or https.
  3. Click Finish.

After adding a Docker Instance, you can build an image. For more information, see Building a Docker Image.

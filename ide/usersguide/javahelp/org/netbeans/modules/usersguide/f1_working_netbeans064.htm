New Project Wizard: Choose Project

You use the New Project wizard to create a new IDE project.

Element Description

Categories

 Displays a list of the categories of projects grouped by technology. This categories in the list are determined by the plugins that are enabled in the IDE.

Projects

 Displays a list of project templates available for the selected project category.

Description

 A short description of the selected project template.

Snapshot Information

Provides snapshot metadata including the following sections:

  • Summary—displays profiler session date and time, snapshot date and time, snapshot location, profiler settings when snapshot was taken.

  • User Comments—displays user comments. These comments may be edited by clicking the Edit button.
  • Settings—displays the profiler settings.

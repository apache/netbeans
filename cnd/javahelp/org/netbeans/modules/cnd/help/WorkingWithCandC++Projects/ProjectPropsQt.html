C/C++ Project Properties Dialog Box: Qt

See Also 

The Qt panel lets you set numerous options for Qt projects, including which Qt modules are used by the project. The IDE displays a tooltip when you mouse over each option.

You must install the Qt framework separately for use by the IDE. The IDE then launches the Qt tools for you as you work with the Qt source files.

Please consult the Qt documentation at http://qt.io/ for more information about the Qt modules, Qt intermediate files, and qmake command.

Git: Select Commits to Rebase Dialog Box

Use the Select Commits to Rebase dialog to forward-port local commits to another destination commit. More

In the Select Commits to Rebase dialog box, you have to set the following properties:

Element Description

Rebase

 Choose a rebasing mode.

Current Branch to Tracked Branch's Head

 Select to move the current branch commits to the head of its tracked branch.

Current Branch to Any Destination

 Select to move the current branch commits to any specified destination.

Rebased Branch

 Displays the branch to rebase.

Destination Revision

 Specify the new base for the selected branch.

Preview

 Displays a sample result of the completed rebase operation.

B (Rebased Branch)

 A currently selected branch.

D (Destination Revision)

 A revision to which rebased revisions are moved.

T (Tracked Branch)

 The tracked branch of the current branch.

Click Rebase to complete rebasing according to the defined settings.

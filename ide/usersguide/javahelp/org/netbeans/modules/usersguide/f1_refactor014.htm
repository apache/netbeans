Move Inner to Outer Level Dialog Box

Use to specify options for the Move Inner to Outer Level refactoring command. Use Move Inner to Outer Level to move an inner class up one level of hierarchy.

Element Description

Class Name

 Displays the name of the class to be moved.

Declare Field for the Current Outer Class

 Generates an instance field for the current outer class and the outer class is passed to the constructor.

Enter a name for the outer class' instance field in the Field Name text field.

Field Name

 Specifies the name of the field to be created if the Declare Field for the Current Outer Class is checked.

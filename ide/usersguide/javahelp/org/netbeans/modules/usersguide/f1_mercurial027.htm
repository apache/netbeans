Mercurial: Clone External Repository Wizard: Mercurial Paths

The Mercurial Clone External Repository wizard enables you to create a local clone of a Mercurial repository.

This panel contains the following elements:

Element Description

Default Pull Path

 This field specifies the path to use when performing a pull if no location is specified. Click Change to modify the path to add a username and password.

Default Push Path

 This field specifies the path to use when performing a push if no location is specified. Click Change to modify the path to add a username and password.

Set Default Values

 Click to use the default values for the paths.

Add Server Instance Wizard: Domain Folder Location

You see this page of the Add Server Instance wizard when you are creating a new domain or specifying an existing local domain of the GlassFish Server v1, GlassFish Server v2.x or Sun Java System Application Server that is not the local default domain. Use this page of the wizard to specify the location of the domain you are registering.

To register an instance of GlassFish Server v1 or v2.x you need to install the GlassFish Server v1, v2.x plugin from the Update Center.

If you are specifying the location of an existing domain, type the path to the domain folder in the Domain Folder text field, or click Browse to use the file chooser to navigate to the domain on your system.

If you are creating a new domain, in the Domain Folder text field, type the path to the folder you want to contain the new domain folder, then type the name of the domain. Alternatively, click Browse and use the file chooser to locate the containing folder and then type the name of the new domain.

For example, type path-to-location/GlassFishDomains/MyDomain to create a domain called MyDomain in the folder GlassFishDomains, where the folder MyDomain does not yet exist. The IDE creates the folder MyDomain for you.

CPU Snapshot Comparison

Use to view a comparison of two CPU snapshots.

Element Description

Call Tree

 Displays a call tree showing the method call chain and the time/number of invocations for executing threads and methods in each context.

Hot Spots

 Displays the total execution time and number of invocations for each method, irrespective of the context.

New C/C++ Project With Existing Sources Wizard: Project Name and Location

See Also 

The New C/C++ Project With Existing Sources wizard enables you to create a project based on an existing makefile.

In the Project Name and Location page, you can set the following properties:

  • Project Name. Specifies the display name of the project. This name is also used to create the project folder.
  • Project Location. Specifies the folder where you want to store the project metadata.

The following properties are displayed:

  • Project Folder. Displays the folder where project metadata is to be stored based on the project name specified.
  • Build Host. Displays the build host that will build the project.
  • Tool Collection. Specifies the tools that will be used on the build host.

New Embedded Application Wizard: Project Name and Location Page

Use to create a new project for a Java ME/MIDP application.

Element Description

Project Name

 Specifies the name of the new project.

Project Location

 Specifies the parent folder where your new project folder is to be stored.

Project Folder

 Displays the location of a project folder that the IDE creates, which is based on the name of the new project

Create Default Package and IMlet Class

 Creates a project with an IMlet class.

If you do not check this box, the IDE creates an empty project instead.

If you are importing a mobile application from another source, Create Default Package and Main Executable Class is not available.

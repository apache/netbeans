Plugins Manager: Downloaded

You use the Plugins manager to dynamically update the IDE from the registered update centers and to manage IDE plugins. You use the Downloaded tab of the Plugins manager to add and install manually downloaded plugins.

The left pane displays the manually downloaded plugins that you can install. Select the checkbox in the install column for each downloaded plugin that you want to install. The right pane displays a description of the selected plugin.

Element Description

Add Plugins

 Click to navigate to the location of a downloaded plugin.

Search

 Type text in the Search field to locate any plugins in the list of downloaded plugins that contain the text in the plugin description.

Install

 Click to install each of the selected downloaded plugins.

Keystores Manager

Use to manage the security keystores for your MIDP project.

Element Description

Keystores

 Displays the existing keystores for the current project. From this window, you can select a keystore to edit.

Keystore File

 Displays the file location of the selected keystore..

Keys

 Displays the keys within the keystore file.

Check the Show Details checkbox to see the details of key pairs within the keystore.

Add Keystore

 Enables you to import existing keystores or add new keystores to the project.

Remove

 Deletes keystores from the project.

Unlock

 Enables you to enter a password to unlock a key pair for use.

Export

 Enables you to export a key pair to an emulator platform.

New

 Creates a new key pair.

Delete

 Deletes a key pair.

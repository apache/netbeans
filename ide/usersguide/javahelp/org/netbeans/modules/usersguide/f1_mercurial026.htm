Mercurial: Clone External Repository Wizard: Mercurial Repository

The Mercurial Clone Repository dialog box enables you to create a local clone of an existing Mercurial repository.

In order to clone a Mercurial repository, you need to indicate the source repository and the destination of the clone (i.e., the name and location of the parent folder, and the name of the clone).

This panel contains the following elements:

Element Description

Repository URL

 Enter a URL that contains the connection protocol and the location of the repository. You can select a connection protocol from the dropdown list.

Select Folder

 Click to locate a directory on your local system. This button is visible when you specify file as the connection protocol.

User

 Enter the user name for the repository, if required. This field is visible when you specify http as the connection protocol.

Password

 Enter a password, if required. This field is visible when you specify http as the connection protocol.

Proxy Configuration

 Click to open the proxy configuration panel in the Options window. This field is visible when you specify http as the connection protocol.

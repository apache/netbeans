Usages Window

Use to display a tree view of files and class elements that contain the Java identifier for which you searched for with the Find Usages command.

You can perform the following operations in the Usages window:

  • Double-click a line to open the file that the line of output refers to in the Source Editor.
  • Click the icon to display any changes you have made since initiating the Find Usages command.

Add Server Instance Wizard: Oracle WebLogic Server Instance Properties

You see this page of the Add Server Instance wizard when you are registering a remote Oracle WebLogic Server instance.

Use this page to specify the properties of the instance that you are registering.

Element Description

Hostname

 Type an IP address of the WebLogic Server instance.

Admin Port

 Type the port number.

SSL/TLS

 Select if data encryption is needed and provided by the server.
Server debug mode enabled Select to set the remote WebLogic server to the debug mode.
Debug Port Specify the port number where debugging is enabled.
Username Type the username.
Password Type the password.

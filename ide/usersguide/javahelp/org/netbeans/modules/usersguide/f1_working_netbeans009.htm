Editor Options: Macros

Use this tab to edit, add, and remove Source Editor macros. It is generally easier to add a new macro by recording it than by adding one manually in this tab.

Use the buttons to perform the following tasks.

Element Description

Macros

 The Macros table displays the current macros and the shortcuts to start the macro.

Macro Code

 The Macros Code text area displays the actions that are performed by the macro that is selected in the Macro table. You can type actions and code in the text area to modify the selected macro.

New

 Click New to open a dialog box where you can enter the name for a new macro.

Set Shortcut

 Click Set Shortcut to open a dialog box where you can set a keyboard shortcut for the macro that is selected in the Macros table.

Remove

 Click Remove to remove the macro that is selected in the Macros table.

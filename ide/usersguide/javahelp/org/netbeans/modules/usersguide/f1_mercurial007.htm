Mercurial: Revert Modifications Dialog Box

The Mercurial Revert Modifications dialog box enables you to revert your working copy to the state of a previous changeset.

Element Description

Choose from Revisions

 Select a revision you want to revert to from the drop-down field. Revisions are displayed containing revision numbers and corresponding changeset IDs in parentheses. Revisions are listed according to date (i.e., the most current revision is displayed by default).

In the Options pane, select the Back up any Locally Modified files to <file>.orig option to prompt the IDE to store a temporary copy of your working copy version prior to reverting changes.

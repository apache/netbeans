Create Test Wizard: Name and Location

You can change the values for the following fields:

  • Test Name: The name to be displayed for the test in the Test Files folder.
  • Folder: The name of the directory or physical folder in which to store the test files. You can type the name or use the Browse button to select the directory.
  • Test File Name: The filename for the test source code.
  • Source Extension: The default is .c for C source, and .cpp for C++ source. You can select a different extension from the list if your build environment requires a different extension.
  • Header Extension: The default is .h but you can select a different extension if your build environment requires a different extension.

Click Finish to create the test files.

Look in the Test Files folder in the project. You must edit these preliminary test files to make useful tests for your project.

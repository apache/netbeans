Tasks: Modify Templates Dialog Box

Use this dialog box to specify the format for the information that is automatically added as a comment to tasks and included in commit messages.

Element Description

Commit Message

 Modify the text in the window to change the format that is used for the commit message.

Position

 Select where to display the text specified in the Commit Message window.

    Above the Message

 Choose to have the text automatically added at the top of the commit message specified in the Commit dialog box.

    Below the Message

 Choose to have the text automatically added at the bottom of the commit message specified in the Commit dialog box.

Task Comment

 Modify the text in the window to change the format that is used for the task comment.

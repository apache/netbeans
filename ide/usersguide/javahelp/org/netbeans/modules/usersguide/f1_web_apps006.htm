web.xml Visual Editor: Welcome Files

Use the Welcome Files section to add, remove, and view your web application's welcome files. A welcome file is a JSP file, HTML file, or servlet that is displayed when the web application is deployed. In the Visual Editor, click Pages at the top of the editor to open the Pages view. Click the Welcome Files header to open the Welcome Files section. How?

Element Description

Welcome Files

 Specifies the path to a welcome file.

Browse

 Click to find and select the welcome file of your choice. You can choose more than one welcome file, but only the first existing welcome file is used as the web application's welcome file. You can register other files as welcome files for testing and debugging purposes.

To define one ore more JSP files, HTML files, or servlets as welcome files, you can register them in the web.xml file.

Before you can register a welcome file in the web.xml file, you must create the JSP file, HTML file, or servlet or import it from your existing sources.

During development time only, you can specify the application's welcome file by right-clicking the project node in the Projects window, choosing Properties, clicking Run, and typing it in the Relative URL text box. Note that the welcome file specified in the Project Properties dialog box is the welcome file that is used first. If no welcome file is defined in the Project Properties dialog box, the first existing welcome file defined in the web.xml file is used.

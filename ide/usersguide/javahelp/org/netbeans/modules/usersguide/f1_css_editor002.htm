Add CSS Property Dialog Box

The Add CSS Property dialog box enables you to add declarations to a CSS rule in a style sheet by adding properties and values. The dialog box displays a list of properties and provides a dropdown list of standard values for each property.

Element Description

Filter

 Type a string in the field to limit the properties that are displayed to properties that contain the string.

Click to display the list of properties organized by category.

Click to displaythe list of properties organized alphabetically.

New HTML5/JS Application Wizard: Site Template

The New HTML5/JS Application wizard creates a new project containing an empty HTML5 application.

In the Site Template page, select one of the following template options.

Option Description

No Site Template

 Select this option to create a project that will only contain an index.html file in the site root.

Select Template

 Select this option to create a project that is based on an HTML5 template. If you select this option you can click Browse to locate the archive on your local system. Alternatively, you can type a URL of an existing template.

You can use the dropdown list to select from a list of templates that you have saved.

Download Online Template

 Select this option to create a project that is based on an online template. If you select this option you need to select a template from the list.

You must be connected to the Internet and your proxy must be properly configured to download an online template.

Click Finish to create a project.

Rename Dialog Box

Rename Refactoring allows you to safely rename a PHP element across all files in an application. For safety, you are required to preview and approve all the changes in the Refactoring window.

Element Description

New Name

 Type the new name.

Rename Also File With the Declaration

 If selected, the file name is also renamed.

    Use Lower Cases for a File Name

 If selected, the name of the renamed file is in lower case characters.

Preview

 Click to preview the changes in the Refactoring window. The Refactoring window opens automatically when you click Preview.

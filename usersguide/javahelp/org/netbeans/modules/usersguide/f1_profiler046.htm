SQL Query Viewer

Use to view the selected query.

The dialog contains the following element for browsing information contained in the SQL query.

Element Description
SQL Query Displays the SQL query text. You can right-click the query text and choose one of the following options from the context menu: Cut, Copy, Paste, or Select All.

Select RichFaces Library Dialog Box

Use to select a registered library that contains the JSF RichFaces 4.0 library JAR files. If the library does not exist, you can click Create Library to create a custom library that contains the required RichFaces libraries.

The dialog displays a list of the RichFaces 4.0 JAR files that the custom library must contain and a list of additional libraries that are recommended for inclusion in the custom library. To register a RichFaces component framework library, you first create a library that contains the required JARs and then select the library from the drop-down list. The IDE will add the library JARs to the project's classpath.

You need to download the RichFaces 4.0 library JARs to your local system before you can create the library. The RichFaces 4.0 JARs are not bundled with the IDE.

Profiling Points Report Window

Use to view details about the profiling point that you select in the Profiling Points window.

The information displayed includes the following details:

  • Type and status of profiling point
  • Location of profiling point, including the project and the line in the source code
  • Number of times the profiling point was hit

The complete set of details is dependent on the profiling point type.

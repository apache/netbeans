Feed Reader Sample

The Feed Reader sample consists of the following projects:

Element Description

FeedReader

 Module project containing the application's functionality.

jdom

 Module project that wraps the JDom library.
rome Module project that wraps the Rome library.

rome-fetcher

 Module project that wraps the Rome Fetcher library.

feedreader-suite

 Module suite project that wraps the other module projects and allows them to be branded and deployed as a single unit.

To open and use the Feed Reader sample:

  1. Choose File > New Project (Ctrl-Shift-N) from the main menu.
  2. Under Categories, expand Samples, and select NetBeans Modules.
  3. Under Projects, select FeedReader. Click Next.
  4. Type a name for the sample.
  5. Click Finish.

Mercurial: Switch Branch In Repository Dialog Box

The Switch Branch In Repository dialog box enables you to switch the branch that your working directory is on to a branch that already exists in your repository.

The Switch Branch in Repository dialog box contains the following elements:

Element Description

Choose from Branches

 Specify the branch you want to switch to.

Filter

 (Optional) Specify your filtering criteria to filter the displayed branches.

Author

 Name of a person who committed the revision.

Date

 The date when a commit was made.

Description

 A message specified during the commit of the revision.

Do a Forced Update

 (Optional) Switch to a selected branch even if your local changes are lost.

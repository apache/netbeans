CVS: Switch to Branch Dialog Box

The CVS Switch to Branch dialog box enables you to switch the selected files or folders in your local working directory to either the trunk or another branch. Revisions are not merged when you switch from a branch to the trunk or from the trunk to a branch.

CVS Status Labels for files in the Projects, Files or Favorites windows can indicate if you are working from the repository trunk or a specific branch. To activate CVS Status Labels, choose View > Show Versioning Labels from the main menu.

Select one of the following actions:

Element Description

Switch to Trunk

 If selected, your local working files are switched to the trunk.

Switch to Branch

 If selected, your local working files are switched to the specified branch. Enter the name of the branch in the text field or click Browse to see a list of the available branches.

Add Java Platform Wizard: Choose Platform Folder

Use to register various versions of the Java Development Kit (JDK) and other Java tools for use in the IDE. Once you have registered a JDK, you can proceed to configure a project to use that JDK for compilation, execution, and debugging (in the Project Properties window).

Element Description

Folder

 Specifies the parent folder of the Java platform.

Platform Folder

 Specifies the platform folder.

File

 (Read-only ) Displays the path to the platform folder.

File Format

 Specifies the format of the platform folder from the dropdown list. The default is Platform Folder.

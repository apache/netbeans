Plugins Manager: Updates

You use the Plugins manager to dynamically update the IDE from the registered update centers and to manage IDE plugins. You use the Updates tab of the Plugins manager to view and install available updates of installed modules.

This tab displays a list of installed modules that have an available update. When you select a plugin in the left pane, the right pane displays a description of the contents of the update.

The left pane displays updates to currently installed modules that were available the last time the IDE checked the update centers.

Element Description

Check for Updates

 Click to check the update center immediately for updates to plugins.

Search

 Type a string in the text field to filter the list of displayed plugins.

Update

 Click to update the selected plugins.

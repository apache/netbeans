Refactoring Inline

With inline refactoring, you can replace macros with actual values. This is useful if a macro is only used one time in a project, for example.

Element Description

In place

 Applies changes only to the selected location.

In file

 Applies changes to item's occurrences in the current file.

In project

 Applies changes to item's occurrences in the current project.

Creating a C or C++ Project

See Also 

  1. Choose File  New Project (Ctrl-Shift-N).
  2. Select the appropriate template for your project. The templates are described in About C/C++/Fortran Projects.
  3. Follow the steps in the remaining pages of the wizard. For more information, see the following topics:
    • New C/C++ Project With Existing Sources Wizard
    • New C/C++ Project From Binary File Wizard
    • New C/C++ Application Wizard
    • New C/C++ Dynamic Library Wizard
    • New C/C++ Static Library Wizard
  • You can create a project from a binary file in several other ways. See Creating a C or C++ Project from a Binary File for more information.

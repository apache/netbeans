Add Java Platform Wizard: General Information Page

Use to define the general parameters for an emulator that is not UEI-compliant.

Element Description

Platform Home

 Specifies the path to the directory where the emulator platform is installed. You can enter a path, or use the Browse button to navigate to the directory.

Platform Name

 Specifies a name for the emulator platform.

Device Name

 Specifies a name for the emulator platform.

Preverify Command

 Specifies the command line syntax that invokes the emulator to preverify a MIDlet.

Execution Command

 Specifies the command line syntax that invokes the emulator to execute a MIDlet.

Debugger Command

 Specifies the command line syntax that invokes the emulator to debug a MIDlet.

To see descriptions of the command line syntax parameters, click in the appropriate field.

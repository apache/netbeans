Tasks: Blocks Task Dialog Box

You use this dialog box when you want to update an task to indicate that the task prevents the resolution of a specific existing task. You open this dialog box by clicking the Search button (...) next to the Blocks text field in an task's form.

Element Description

Select task that this task blocks

 Enter the id or text contained in the summary of the task that is blocked.

You can also select a task from the dropdown list of tasks that you recently viewed in the IDE.

JavaMail Session Properties

You see this page of the New File wizard when you are defining a JavaMail Session resource.

This page of the wizard enables you to add additional configuration information and properties for a JavaMail Session resource.

Click Add to add a row for the new property to the table. In the Properties table, enter the Name and Value of the new property and click Enter to save the changes to the property table. To remove a property, select the property in the Properties table and click Remove.

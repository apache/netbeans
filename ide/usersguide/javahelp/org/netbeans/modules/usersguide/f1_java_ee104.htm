Web Service Attributes Editor: WSDL Customization: Bindings

Use the Bindings section to customize the attachment mechanism that provides additional metadata as to how the WSDL type is bound to Java.

Element Description
Enable MIME Content When deselected the data will be bound to a Byte array. When selected wsimport will use the information in the mime:content portion of the binding section and bind it to java.awt.Image.

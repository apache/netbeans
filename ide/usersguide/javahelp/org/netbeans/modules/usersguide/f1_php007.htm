New PHP Project Wizard: PHP Frameworks

From this page you can select an optional framework to use with your PHP project. When you select a framework, options appear for configuring the IDE to use that framework.

Element Description

Options

 Click this link to open the Framework & Tools tab in the PHP category of the Options window. You can use the Options window to configure the global options for the selected framework.

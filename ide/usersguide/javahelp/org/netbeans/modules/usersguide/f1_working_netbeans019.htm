Options Window: Team: Tasks

You use this panel to configure general properties for how the Tasks window is synchronized with registered task repositories.

You can add task repositories in the Services window or the Tasks window.

Element Description

Task Repositories

 Select a task repository from the list to modify the properties of that repository. The default task repository is Bugzilla.

    Opened tasks refresh insterval

 Specify how often the IDE checks for changes to open tasks.

    Opened queries refresh interval

 Specify how often the IDE checks open queries for updates.

Tasks synchronize interval

 Specify how often the tasks in the Tasks window is updated.

Tasks Limit

 When selected you can limit the number and type of tasks that are displayed in the Tasks window.

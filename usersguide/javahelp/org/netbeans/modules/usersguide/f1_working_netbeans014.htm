Manage Keymap Profiles Dialog Box

You use this dialog box to create, remove, import and export keymap profiles.

This panel contains the following elements.

Element Description

Profiles

 Select a profile from the list.

Duplicate

 Type in the text field to filter the shortcuts by the name of the command.

Restore Defaults

 Click to restore the selected profile to the default settings. This button is disabled if the selected default has not been modified.

Delete

 Click to delete the selected profile. You cannot delete the default profiles that are included with the IDE.

Export

 Click to save the selected profile as an XML file on your local system.

Import

 Click to locate a saved profile on your local system.

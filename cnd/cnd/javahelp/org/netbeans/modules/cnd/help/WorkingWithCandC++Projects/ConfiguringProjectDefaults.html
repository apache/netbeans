Configuring C and C++ Default Project Settings

See Also 

You can configure default settings, which apply to all new C and C++ projects you create, by using the Options window. You can also configure settings for a particular project by editing the Project Properties for the project.

To configure C and C++ default project settings:

  1. Choose Tools > Options.
  2. Click C/C++.
  3. See Options Window: C/C++ for information about the options.

To override the default settings for an individual project:

  • Right-click the node for the project in the Project window and select Properties.

See Setting Project Properties for more information.

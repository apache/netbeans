Options Window: Team: Action Items

You use this panel to specify the character patterns that are used for identifying action items. Action items are listed in the Action Items window.

You can set the following action item options:

Element Description

ToDo Patterns

 The text files displays the tag strings used in the source code to identify action items. Use the buttons to the right of the text field to add, delete or modify the ToDo patterns.

Show ToDos from comments only

 If selected, ToDo patterns that are outside of comment blocks are ignored.

Supported File Types

 Lists the file types that can are scanned for action items. Click Add to specify a new MIME type.

Comment Patterns

 For the selected file type you can enable the character patterns that are used to specify comments that are scanned.

    Scan Line Comments

 If selected, line comments that are identified by the specified character pattern are scanned for action items. This option is enabled by default.

    Scan Block Comments

 If selected, block comments that are identified by the specified character pattern are scanned for action items. This option is enabled by default.

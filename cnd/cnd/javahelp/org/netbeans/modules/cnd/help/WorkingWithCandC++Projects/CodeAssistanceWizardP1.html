Configure Code Assistance Wizard: Select Mode

See Also 

For a C/C++ Project With Existing Sources, the Configure Code Assistance wizard lets you specify how your code will be parsed by the built-in parser used for the code assistance features of the IDE.

The Select Mode page lets you choose how the wizard determines the correct parsing options (include files and macro definitions) for your project.

  • Simple Mode. If the project has been built with debug information, the wizard asks you for minimal information to use in determining the correct parsing options.

  • Advanced Mode. If the project has not been built with debug information, the wizard asks you for more information to use in determining the correct parsing options.

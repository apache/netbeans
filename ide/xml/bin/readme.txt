This folder contains scripts used while development, not for module runtime.

* gen-icons_xml.sh
- It is used for generating xml/www/*/data/icons.xml files.

* gen-l10n_list.sh
- It is used for generating xml/*/l10n.list files.

* init-xalan.sh
- Extends classpath with JAXP, Xerces and Xalan.

* update_www.sh
- It is used as automatical update of XML web pages.

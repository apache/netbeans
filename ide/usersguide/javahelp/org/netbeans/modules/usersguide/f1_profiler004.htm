OQL Console: Save Query Dialog Box

Use to save an OQL query and to assign a name and description to the query. Saved queries are listed in the Saved Queries pane of the OQL Console tab in the Heap window. ?

Element Description

Create a New Query.

 Saves the query in the Query Editor pane as a new named query and enter the following details about the query.

Name

 Specifies a name to identify the query.

Description

 (Optional) Enables you to provide a short description of the query.

Use Existing Query

 Modifies an existing custom saved query. When you select this option you must select a custom query from the list. The query in the Query Editor pane replaces the query associated with the custom query that you select.

This option is only available if custom queries exist.

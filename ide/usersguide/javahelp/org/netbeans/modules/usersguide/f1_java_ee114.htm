Add Server Instance Wizard: JBoss Server Instance Location

You see this page of the Add Server Instance wizard when you are registering a JBoss Application Server instance. Use this page of the wizard to specify the location of the instance you are registering.

Type the location of the server instance in the Installation Location text field, or click Browse to use the file chooser to navigate to the instance on your system.

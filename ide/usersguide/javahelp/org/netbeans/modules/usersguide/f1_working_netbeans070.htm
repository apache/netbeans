Login to netbeans.org Dialog Box

This dialog box opens when you choose to submit an task with NetBeans IDE and you are not logged in to netbeans.org.

You can report a problem with NetBeans IDE and submit an task to the netbeans.org task repository. To report a task from the IDE you need to have an account at netbeans.org and you must be logged in to that account in the IDE. After you log in, you can submit a task in the New Task window.

Element Description

Username

 Type your netbeans.org username.

Password

 Type your netbeans.org password.

If you do not have an account, you can click the link in the dialog box to open netbeans.org in your browser and create an account.

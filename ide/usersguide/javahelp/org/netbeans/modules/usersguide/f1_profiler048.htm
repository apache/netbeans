Edit Filter

Use to edit a filter.

The dialog contains the following elements for editing the details of a filter.

Element Description
Name Specifies the name of the filter.
Color When selected, the custom color for the matching results can be defined by clicking the Default color button. When deselected, the matching results are displayed using the default color.
Value Specifies the value of the filter.

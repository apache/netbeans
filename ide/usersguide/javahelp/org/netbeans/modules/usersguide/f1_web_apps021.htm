Standard Web Project Properties Dialog Box: Disabled JAX-RPC Web Service Clients

Use to preconfigure the features that are used by the wscompile tool to compile a JAX-RPC web service client.

If you have not created any JAX-RPC web service clients, this page is empty.

Note: This page is for JAX-RPC (J2EE 1.4) web service clients only. Even though the project does not contain JAX-RPC web service clients, it could contain JAX-WS (Java EE 5) web service clients.

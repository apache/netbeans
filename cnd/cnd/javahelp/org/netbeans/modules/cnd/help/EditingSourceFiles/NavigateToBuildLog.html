Navigate to Build Log from a Compiler Hint

If your code shows compiler hints or warnings and displays a light-bulb icon in the left margin, you can easily navigate to the build log for more information about the warning.

To jump to the build log for details about a compiler warning:

  • Place the mouse cursor on the affected line and press Alt+Enter.
  • When the popup menu displays, select "Show in Output" for the compiler warning you want to investigate.

 

The IDE shows the Output window displaying the build log and scrolls to the matching compiler output for the compiler warning.

Options Window: PHP: Frameworks & Tools: Zend

In this panel you can set global options for projects that use the Zend framework.

Element Description

Zend Script

 Specify the path to the Zend Framework run script and default parameters. Use the Browse and Search buttons to locate Zend run scripts on your local file system.

Register Provider

 If you are using Zend Framework 1.10 or newer, register the provider. Look at the output window after you click Register Provider. The IDE displays the result of registering the provider in the Output windows. If registration failed, you may have a problem in your zf.php or Zend Framework run script.

Enterprise Application Client Project Properties Dialog Box: Run

Use the Run page to configure options for running your project. You can set the main class, program arguments, and working directory for program execution. You can also specify a space-separated list of VM options in this panel.

Notes:

  • The Browse button for the Main Class field only shows classes in the project's source folders. If you want to specify a class in one of the JAR files on the project's classpath, type the fully qualified class name in the Main Class field.
  • Arguments are passed only to the main class, not to individual files.

Wizard Wizard: Name, Icon, and Location

The New Wizard wizard creates a new Wizard.

You open the New Wizard wizard by right-clicking a module project, choosing New > Other and selecting Wizard from the Module Development category.

Element Description

Class Name Prefix

 Specifies the name of the new panels and action class.

Package

 Specifies the name of the package where the class will be housed.

You can set the following items only if the wizard will be registered as a New File wizard:

Element Description

Display Name

 Specifies the display name for the new wizard in the New File wizard.

Category

 Specifies the New File wizard category where the new file wizard will be located.

Icon

 Specifies the icon that will be displayed in the New File wizard.

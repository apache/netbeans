Module Suite Project Properties Dialog Box: Splash Screen

You open the Module Suite Project Properties dialog box by right-clicking a module suite project and choosing Properties.

The Splash Screen page specifies the splash screen that will be displayed when an application built on top of the NetBeans Platform is started.

Element Description

Progress Bar

 The properties of the progress bar are as follows:
  • Enabled. Specifies that a progress bar will be shown during startup.
  • Color. Specifies the color of the progress bar.
  • Positioning. Specifies the position of the progress bar, in pixels.

Running Text

 The properties of the startup text displayed in the splash screen are as follows:
  • Size. Specifies the size of the startup text.
  • Color. Specifies the color of the startup text.
  • Positioning. Specifies the position of the startup text, in pixels.

Splash Screen

 Specifies an image that will serve as the splash screen.

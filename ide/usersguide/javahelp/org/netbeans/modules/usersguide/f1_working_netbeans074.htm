Notifications Window

The Notifications window displays a list of the IDE errors and warnings that occurred in the current IDE session. You can select a notification in the list to view details about the notification in the right pane of the window.

You can click the column heading in the table to sort the list of notifications. You can also filter the list by string or notification type.

Element Description

Click to filter the list according to a string.

Click to select a filter from the drop-down list and apply the filter to the list. You can also create or edit a filter.

When you encounter problems with the IDE you can use the Notifications window to report problems to the NetBeans development team.

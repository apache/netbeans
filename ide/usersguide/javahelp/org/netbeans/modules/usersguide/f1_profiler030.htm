Details of <Process> Dialog

Provides metadata for the selected external process. It includes the following sections:

  • PID—process ID.
  • Main Class—the main class to be executed.
  • Arguments—the arguments of the main class to be executed.

  • JVM Arguments—Java Virtual Machine arguments, if any.

  • JVM Flags—Java Virtual Machine Flags, if any.

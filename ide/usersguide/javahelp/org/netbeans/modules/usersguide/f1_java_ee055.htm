Visual Editor: Edit Principal

Use to edit the name of a principal assigned to a security role.

This dialog box enables you to edit the following properties:

Element Description

Principal Name

 Modifies the name of the principal assigned to a security role.

The name of the principal must match a name specified on the GlassFish application server.

Class Name

 Modifies the implementation class.

Favorites Window

The Favorites window enables you to access any file on your computer or network without having to create a project for the file.

When you first open the Favorites window, it only contains your computer's home directory. You can add any folder as a top-level folder to the Favorites window by right-clicking a folder in the window and choosing Add to Favorites or by right-clicking in the whitespace in the window and choosing Add to Favorites to open the Add to Favorites dialog box.

Project-level commands, such as Run Project, are not available from the Favorites window. For such commands to be available, you need to have a project open in the Projects window.

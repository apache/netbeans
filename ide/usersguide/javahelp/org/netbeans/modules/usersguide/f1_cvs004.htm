CVS Import Wizard: Folder to Import

The CVS Folder to Import panel of the CVS Import wizard enables you to:

  • specify the folder or project to import from your working directory, and
  • designate a target location in the repository.

When importing into the repository, you specify the following:

Element Description

Folder to Import

 Enter the location of the project, or click Browse to locate the project you want to import.

Import Message

 Enter a message describing the project you are importing.

Repository Folder

 Enter the name of the folder in the repository where you want to import the project. Click Browse to choose from a list of repository folders in the repository.

Checkout After Import

 If selected, the project you are importing into the repository will be checked out to your local working directory after it is imported.

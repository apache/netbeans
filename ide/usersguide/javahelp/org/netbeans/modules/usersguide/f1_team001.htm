New Team Project Wizard: Name and Licenses

The New Team Project wizard enables you to create a project on a registered Team Server.

This wizard enables you to create a project on a Team Server from within the IDE. When you create the project, the IDE sends the project information to the Team Server to create the project and Team Server services are assigned to the project.

Note: You must be logged in to a Team Server to create a Team project.

In the Name and Licenses page you need to set the following properties.

Element Description

Logged in to

 If you are logged in, displays the current Team Server and the user name. Click Login if you are not logged in to open the log in dialog box.

Team Project Name

 The name that is used for the project URL on the Team Server website.

Project Title

 The display name of the location. This name is used to identify the project in the IDE and the project page on the Team Server website.

Project Description

 (optional) A description of the project. You can add or modify the description on the project webpage.

Project License

 Select a license from the drop-down menu. You can modify the license on the project webpage.

Add Catalog Dialog Box

Use this dialog to add an existing XML Schema and DTD Catalog to your project.

The Add Catalog dialog contains the following fields:

Element Description

Catalog Type

 Select the catalog type from the drop-down box. See XML Catalog Formats for more information.

Catalog URL

 Location of the existing DTD or XML Schema catalog that you want to add to the project.

    Browse

 Browse for the catalog in your local file system.

Prefer Public ID

 Select if the catalog resolver should try to use the PUBLIC identifier before resorting to the SYSTEM identifier. Selected by default.

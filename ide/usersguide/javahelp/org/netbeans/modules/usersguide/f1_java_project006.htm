Free-Form Project Properties Dialog Box: Build and Run

Use to add map IDE commands to targets in your free-form project's Ant script.

The combo box for each command (Build Project, Clean Project, Generate Javadoc, Run Project, and Test Project) contains all of the targets in your project's Ant script.

Add shortcuts to any of your Ant script's targets to the project node's context menu in the Mapping of Custom Contextual Menu Items panel. Click Add to add a row to the panel. Click in the Label column and add a shortcut label. Then click in the Ant Target column and select the desired target from the drop-down list.

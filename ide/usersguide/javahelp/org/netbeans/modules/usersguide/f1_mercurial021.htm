Mercurial: Manage Tags Dialog Box

Use the Manage Tags dialog box to browse and delete tags. More

You access this dialog box by selecting a versioned project, file or folder and either:

  • choosing Team > Branch/Tag > Manage Tags from the main menu
  • right-clicking the selected item and choosing Mercurial > Branch/Tag > Manage Tags

In the Manage Tags dialog box, the following properties are available:

Element Description

Tag Name

 Name of the tag selected in the Tags list to the right.

Tagged Commit

 Displays the details of the tagged commit.

Revision

 Name of the revision that is tagged.

Details

 Displays the details of the revision.

Author

 Name of a person who committed the tagged revision.

Date

 Date when the commit of the revision was made.

Description

 Message specified during the commit of the revision.

Tags

 Select a tag to be reviewed, updated, or removed.

Click Remove to delete a tag, Update to to open the Update Repository dialog box with the preselected revision, or OK to close the dialog box.

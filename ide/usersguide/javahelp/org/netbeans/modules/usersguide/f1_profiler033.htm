Profile Class / Profile Method (Select Feature) Dialog

Use to open a profiling session for either methods or classes profiling.

Element Description

Select the project to profile:

 Choose the project from the drop down menu.

Profile project

 Use to launch the profiler with the select project.

Attach to project

 Use to attach to an already running project.

Select the feature to handle the action:

 (Optional) Use to select a feature in case multiple profiling features can handle the action.

Methods

 Use to profile performance of the selected class or method.

Objects

 Use to profile memory footprint and/or allocations of the selected class.

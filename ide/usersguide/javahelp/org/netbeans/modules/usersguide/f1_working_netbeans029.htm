Tasks: Duplicate Task Dialog Box

You use this dialog when you want to resolve a task and want to mark the task as a duplicate of an existing task. When you select Resolved as the task's Status and select Duplicate as the Resolution, the IDE displays a text field for the id of the task that is duplicated.

Element Description

Select the duplicate task

 Enter the id or text contained in the summary of the duplicated task. You can also select an task from the dropdown list of tasks that you recently viewed in the IDE.

Options Window: Miscellaneous: Diff

You use this panel to specify the Diff Engine to use when comparing file revisions. If you use an external diff engine you can set diff command options in this panel.

You can select the following diff engine options:

Element Description

Internal Diff

 If selected, the IDE uses its internal diff engine. When using the internal diff engine, you can let the engine ignore white spaces and changes in case. The former option is selected by default.

External Diff

 If selected, the IDE uses an external command-line diff executable. You can specify any diff command options (for example, diff -b) to be used when running the diff command from a command-line executable in the IDE. Diff options are not availabe when using the built-in diff engine.

To use this option, you must have a command-line diff executable installed on your system if you want to compare revisions.

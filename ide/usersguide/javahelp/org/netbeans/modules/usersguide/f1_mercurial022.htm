Mercurial: Select Revisions to Diff Dialog Box

Use the Select Revisions to Diff dialog box to compare file revisions. More

You access this dialog box by selecting a versioned project, file or folder and either:

  • choosing Team > Diff > Diff to Revision from the main menu
  • right-clicking the selected item and choosing Mercurial > Diff > Diff to Revision

In the Select Revisions to Diff dialog box, the following properties are available:

Element Description

Local Changes to Base

 .Choose to compare local modifications to the base revision.

Local Changes to Revision

 .Choose to compare local modifications to any specified revision.

Base to Other

 .Choose to compare the base revision to any specified revision.

Click Diff to display the difference between the specified revisions in the Diff Viewer.

Subversion: Select Template Dialog Box

The Subversion Select Template dialog box enables you to select a message template for a commit message.

Element Description

Automatically fill into commit message

 Select to have the specified template message inserted as a commit message in the Commit dialog box during the commit.

Use the buttons to perform the following tasks.

Element Description

Load File

 Click to display the Select Template File dialog box where you can browse for a required file the message template and load it into the Template area of the Select Template dialog box.

Save as File

 Click to save the message template from the Template area of the Select Template dialog box into a file on your system.

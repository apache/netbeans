New JDBC Resource: Additional Properties

This page of the wizard enables you to add additional configuration information and properties for a JDBC resource.

This panel of the wizard contains the following elements.

Element Description

Properties

 This table displays a list of the properties of the JDBC resource.

Add

 Click Add to add a row to the table that you can edit to specify the new property.

Hit the Enter key after you add a value to the table to save the value.

Remove

 Click Remove after you select a row in the table to remove the property.

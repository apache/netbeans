Select Snapshot to Compare Dialog Box

Use to select snapshots that are located on your local system for the purpose of comparing two snapshots. Click OK to view a comparison of the snapshots in the Snapshot Comparison window.

When you invoke this dialog box from the main menu the dialog box displays two text fields. Click Browse to locate the two snapshots that you want to compare.

When you invoke this dialog box from an open snapshot, select a snapshot to compare to the open snapshot. Select one of the following options and then specify a snapshot.

Element Description

From Project

 Selects a snapshot that is saved in the project directory.

The dialog box displays the snapshots that can be compared to the open snapshot and that are located in the default project directory.

From File

 Selects a snapshot that is saved on your local system. Click Browse to locate the snapshot.

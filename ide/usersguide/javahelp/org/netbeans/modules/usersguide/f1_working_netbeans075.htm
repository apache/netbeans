Tasks Window

The Tasks window provides an organized overview of tasks that are recorded in a task repository.

The Tasks window contains the following elements:

Element Description

Filter

 Type a string in the text field to filter the list of tasks to tasks that contain the string.

Categories

 Tasks are grouped by category in this section. You can create custom categories for your tasks. By default this section contains categories for scheduled tasks and recently opened tasks.

    Schedule

 Categories for tasks that have a scheduled completion date are created by default. The default categories are Today, This Week and All.

    Recently Opened

 Default category that groups all recently opened tasks.

Repositories

 Local and remote repositories are grouped in this section. You can add remote task repositories in this section or in the Services window.

    Local Tasks

 Default task repository that you can use to store tasks locally. No external task repository is required.

Add/Edit Push Registry Entry

Use to add or edit a push registry entry for a project configuration.

Element Description

Class Name

 Specifies the MIDlet's class name.

Sender IP

 Specifies a valid sender that can launch the associated MIDlet.

If the value is the wildcard (*), connections from any source are accepted.

If datagram or socket connections are used, the value of Allowed Sender can be a numeric IP address.

Connection String

 Specifies connection string that identifies the connection protocol and port number.

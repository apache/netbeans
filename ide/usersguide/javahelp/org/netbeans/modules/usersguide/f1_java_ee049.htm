Add Server Instance Wizard: Oracle WebLogic Server Instance Location

You see this page of the Add Server Instance wizard when you are registering an Oracle WebLogic Server instance. Use this page of the wizard to specify the location of the instance you are registering.

Element Description

Server Location

 Type the location of the server instance. Use the Browse button to use the file chooser to navigate to the instance on your system.

Domain Registration Type

Local Domain

 Choose to register a local server instance.
Remote Domain Choose to register a remote server instance.

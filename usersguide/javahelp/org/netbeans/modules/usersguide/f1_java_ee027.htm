Standard EJB Module Project Properties Dialog Box: Packaging

The Packaging page lets you set up the filter for which files are included in the EJB JAR file.

Note: To manage the project's classpath, click Libraries in the left panel of the dialog box. To set command-line options for javac, click Compiling.

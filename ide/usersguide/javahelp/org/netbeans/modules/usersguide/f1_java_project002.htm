Add Java Platform Wizard: Platform Name

Use to specify a name for the platform that you can use to identify the platform in the IDE. You can also use this panel to specify the locations of the source files and Javadoc for the new Java platform.

Element Description

Platform Name

 Specifies the display name of the platform.

Platform Sources

 Specifies the location of platform sources.

Type the location of the sources or click Browse to navigate to the location.

Platform Javadoc

 Specifies the location of the platform Javadoc. Type the location of the Javadoc or click Browse to navigate to the location.

Choose Heap Dump Destination Dialog Box

Use to specify where you want to save a heap dump.

Element Description

Profiled project

 Saves the heap dump to the default location in the directory for your project.

This destination is selected by default.

Custom directory

 Specifies a location on your local system for the heap dump.

Click Browse to navigate to a location.

Module Project Properties Dialog Box: Documenting

You open the Module Project Properties dialog box by right-clicking a module project and choosing Properties.

Element Description

Javadoc Title

 Sets javadoc.title in the project.properties file. Note that the IDE cannot build Javadoc unless you mark at least one package as public in the API Versioning page.

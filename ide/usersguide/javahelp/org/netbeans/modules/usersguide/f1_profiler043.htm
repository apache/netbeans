Methods Snapshot Dialog

Use to view and analyze data from an existing Methods snapshot.

The snapshot toolbar contains the following options:

Element Description

Export Selected Snapshot

 Use to export the snapshot. The options that dynamically appear when you select this icon are based on the selected View options.

Compare with another snapshot

 Use to compare two snapshots. When you click on this icon a Select Snapshot to Compare dialog with a list of your snapshots is displayed.

View

Forward Calls

 Use this icon to view forwards calls.

Hot Spots

 Use this icon to view hot spots.

Reverse Calls

 Use this icon to view reverse calls.

Select Threads

 Use this icon to narrow the session scope to specific threads.
Aggregation Use the dropdown to change the granularity of the displayed results.

Methods

 Displays results according to the fully-qualified method names

Classes

 Displays results for all methods of the same class aggregated in a single entry.

Packages

 ll methods of classes that belong to the same package are aggregated

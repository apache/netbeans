Options Window: General

You use this panel to set the default web browser for the IDE and to specify general proxy settings.

You can set the following general IDE options in this panel.

Element Description

Web Browser

 The web browser that the IDE uses for displaying web content. The dropdown list contains a list of available web browsers that are installed on the system and the following options.

    <Default System Browser>

 This browser is your computer's default web browser.

    Embedded WebKit Browser

 This browser is bundled with the IDE and is based on the WebKit browser.

Proxy Settings

 Specify the general proxy settings the IDE uses to access the Internet by choosing one of the following options.

    No Proxy

 When selected, no proxy settings are applied.

    Use System Proxy Settings

 When selected the IDE uses the proxy settings that are used by the local system.

    Manual Proxy Settings

 When selected you must specify the HTTP Proxy and Proxy Port. The HTTP Proxy is the host name or numeric IP address of the proxy server to use to connect to HTTP and FTP sites. The Proxy Port is the port number of the proxy server. Click More to configure more advanced proxy settings.

    Test Connection

 Click to test the proxy setting.

Usage Statistics

 Determines whether anonymous usage data will be sent to the NetBeans team to improve the IDE.

Mercurial: Clone External Repository Wizard: Change Path

The Change Path dialog box enables you to add a username and password to the default push or pull path for the clone of a Mercurial repository.

This panel contains the following elements:

Element Description

Repository URL

 This field specifies the repository URL. Select a protocol from the dropdown list to change the default path.

User

 Type your username for the repository.

Password

 Type your password for the repository.

Clear Path

 Removes the URL from the default path and closes the dialog box.

Set Path

 Modifies the default path to add the user and password and closes the dialog box.

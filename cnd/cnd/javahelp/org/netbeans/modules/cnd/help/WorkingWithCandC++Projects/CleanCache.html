Cleaning Cache Directory

See Also 

If the code assistance feature is undesirable, it might be due to the result of corrupted cache. Cleaning the cache directory might resolve the problem with code assistance.


To Clean the Cache Directories:

  1. Click on Tools and select Options.
  2. Click C/C++ in the top pane of the window.
  3. Click the Code Assistance tab.
  4. On the Code Assistance tab, click the Clean C/C++ cache and restart IDE option to clean the cache directories, and restart IDE.

To include extra files in code assistance, see Configuring Code Assistance For Projects with Existing Sources.

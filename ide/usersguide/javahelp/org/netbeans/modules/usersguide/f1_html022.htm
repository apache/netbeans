Project Properties Window: JavaScript

You can use this panel to switch between JavaScript versions for your project and enable comments in JSON files.

This panel contains the following elements:

Value Description

ECMA Version

 Select the version of the ECMAScript to switch between JavaScript language versions that are more compatible with the targeted browser.

Allow comments in JSON files

 Select to have the IDE ignore comments in JSON files and not to report comments as errors.

Overview of C/C++ Unit Tests

The following table outlines the basic process of working with unit tests in C and C++ projects.


Create Tests

 

  • Generate tests for existing source files.

Edit Tests

 

  • Modify the generated tests to suit your project.

Run Tests

 

  • Run tests for an entire project.
  • Run the test file for a specific source file.

Debug Tests

 

  • Debug tests.

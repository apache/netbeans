Web Service Attributes Editor: Quality of Service: Add Header

In the Add Header dialog box you can add a header element to the list of message parts of an incoming or outgoing message that require protection or security. The dialog box contains the following options.

Element Description

Header

 Select a header from the drop-down list. You can select All Headers to add all the header elements to the list of message parts.

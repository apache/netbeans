Project Properties Window: Cordova

You can use this panel to modify the Cordova properties for your project that are save in the config.xml file of the project.

This panel contains the following elements:

Value Description

Application

 Select this tab to specify the details for identifying the application.

    Application ID

 Type a unique identifier for the application in the text field.

Use reverse domain syntax (com.coolappz.HTML5Application) to ensure the widest possible platform support.

    Application Name

 Type in the text field to modify the name of the application.

    Version

 Type the version of the application.

Plugins

 Select this tab to select Cordova plugins to add to your project.

Mobile Platforms Setup

 Click to open the Mobile Platforms tab in the Options window.

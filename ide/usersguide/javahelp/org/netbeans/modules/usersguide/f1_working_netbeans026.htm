Options Window: Appearance: Look and Feel

You use this panel to customize the look and feel of the IDE.

You can set the following options:

Element Description

Preferrred Look and Feel

 Select a look and feel option from the list. The IDE uses platform-specific user interface components if the components offer sufficient functionality.

    Maximize use of native look and feel

 If selected, the IDE uses native user interface components (for example, native file chooser on OS X).
